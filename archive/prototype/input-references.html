Input Bar — References & Mentions

How the floating input handles templates, emotions, models, images, and more

Two trigger patterns:
/ — Slash commands for actions: /template /model /emotion /style
@ — Mentions for referencing things: @sophia @gym-selfie @previous

How it works: Type / or @ and a floating menu appears with suggestions. Pick one and it becomes a colored chip in the input. The AI knows what you referenced and uses it in generation.

Reference System Summary
One floating card that shapeshifts based on mode and context:

Chat mode — simple text + chips from / and @ commands
Image mode — reference thumbnails above + aspect ratio/count chips in toolbar
Video mode — similar to image but with duration chip
Voice mode — script editor + voice card + emotion tagger

The toolbar stays consistent — left side has tools/context, right side has mode chips + send. This anchors the UI no matter how the top of the card transforms.

/template   /model   /emotion   /style   /ref — color-coded by type so you can scan at a glance.
