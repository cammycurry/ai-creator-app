From Input Bar

From Sidebar
"Create Content" button — direct access to full studio
From Content Detail
"Make Video" / "Make Carousel" on existing content

Studio Changes

Stays the Same

  • Full-screen overlay
  • Left panel: Library (refs + public + recent)
  • Right panel: Type pills + prompt + config
  • Per-type config (carousel slides, video source, voice picker)
  • Results view with iteration
  • All 4 content types

Already Updated

  • Account-level refs (not per-creator) ✓
  • Public library is real (not "coming soon") ✓
  • Simplified reference types ✓

Future (Agent Layer)

  • Chat-style interface replaces/augments input
  • AI selects refs, configures settings
  • Approve/reject generation proposals
  • Iterative conversation about content

How They Work Together

Input bar → type prompt + pick type → generates directly into content grid
Input bar → "Open studio" → studio opens with prompt pre-filled
Studio → attach refs, configure, generate → results view → content lands in grid
Content card → click → detail view → "Make Video" / "Remix" → studio opens with context
All paths call the same generation engine functions.
