Generation Engine (Agent-Ready)

Both the input bar and Content Studio call the same structured generation functions. This means when we add the AI agent layer later, it calls the exact same functions.

generatePhoto(creatorId, prompt, refs?, aspectRatio?, count?)
generateCarousel(creatorId, formatId, slideCount, instructions?, slideEdits?)
generateVideo(creatorId, prompt, source, duration?, ratio?)
generateTalkingHead(creatorId, script, voiceId, setting?, duration?)
// Same functions called by: Input Bar, Content Studio, AI Agent (future), Public API (future)
