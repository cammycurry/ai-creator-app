How should the creation flow work? Pick the approach that feels right.

Full page grid of all templates (carousels + singles). User browses, picks one, then customizes in a side panel that slides out. References attach per-slide in the panel.

B

Split Panel Studio

Two-panel page: left side is the template library + reference library (tabbed), right side is a live "build area" showing slide-by-slide preview as you configure. Drag references from left onto slides on right.

Stepped Flow (Full Page)

Linear full-page flow: Browse → Customize → Generate → Review. Each step is its own screen. Like the Creator Studio wizard but for content.

Key difference

A = everything on one page, side panel for details (like Figma's component library)
B = always-visible split view, drag-and-drop references (like a DAW or video editor)
C = guided steps, one thing at a time (like the Creator Studio wizard you already have)
