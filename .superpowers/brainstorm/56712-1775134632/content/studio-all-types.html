Key Design Decisions

Refs are visual thumbnails inline. When you select a bedroom ref, you see the bedroom thumbnail right in the input area. Not a text tag — a real image.
Sidebar = Your World + Public Library + Recent. Account-wide refs (shared across creators), public curated refs, and your last generations. All in one compact panel.
Video has 3 clear source cards. "From Text" / "From Photo" / "Motion Transfer" — each is a visual card, not a dropdown. Motion transfer explains itself.
Carousel auto-matches refs to slides. Select gym ref → it auto-appears on gym-related slides. "(auto)" badge. User can override.
History = click to reload. Click any recent generation → loads all settings (type, prompt, refs, config) back into the studio. Regenerate with tweaks.
