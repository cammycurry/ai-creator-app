## What Needs Work

### Content Studio Gaps (HIGH PRIORITY)
1. **AI Assistant** — no AI help in the studio. Need: prompt suggestions, script writing, content ideas, prompt improvement. "Suggest" button next to prompt → Grok/Flash call.
2. **Generation History** — no way to see/reload past generation configs. Every generation should auto-save settings. History panel lets users click to reload and regenerate.
3. **Visual Polish** — studio is functional but needs design attention to feel premium.

### Reference System Gaps
4. **Account-level references** — currently all refs are creator-scoped. Need `creatorId` optional for account-wide refs (your bedroom shared across all creators).
5. **Public reference library** — "Library" tab shows "Coming soon". Need user-facing query of `PublicReference` model + admin curation pipeline.
6. **Reference generation** — users should be able to generate reference images (not just upload), iterate on them.

### Other
7. **Content Studio mobile** — reference panel needs to become a bottom drawer. Test on real devices.
8. **Content Presets** — save/load factory configurations (preset model + UI).
9. **S3 cleanup** — `deleteReference` doesn't delete the S3 object. Add `deleteImage` to s3.ts.

## Decisions Made

- **Kling 3.0 for all video** — best character consistency, fast, reasonable cost
- **Kling lip sync** ($0.014/unit) instead of Hedra (not on Fal.ai) — much cheaper than expected
- **ElevenLabs Turbo v2.5** for voice — industry standard quality
- **References are the universal input** — same refs work for all content types
- **Factory model** — studio is a content factory, content types are product lines, configs are saved
- **Floating input stays** for quick photo generation — studio is the full experience
