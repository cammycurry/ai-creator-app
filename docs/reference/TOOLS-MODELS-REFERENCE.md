# Tools & Models Reference

> Extracted from prototype catalog. These are the AI models and tools available in the engine.

## Tools
- **Consistent Character** — Keep character identity across scenes using ref images
- **Background Swap** — Replace background in any image with AI
- **Face Swap** — Swap faces between images/videos
- **Image Upscaler** — Enhance images to 4K with AI upscaling (Lupa/Topaz style)
- **Remove Background** — Clean cutout with transparent bg
- **Lip Sync** — Add realistic speech to videos (Pixverse/HeyGen)
- **Voice Clone** — Clone a voice from audio sample (ElevenLabs)
- **Image to Video** — Animate any still image into video
- **Outfit Swap** — Change clothes while keeping identity
- **Multi-Angle** — Generate multiple camera angles of same person
