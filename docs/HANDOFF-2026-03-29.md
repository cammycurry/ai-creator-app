### 1. Carousel/Content Creation Studio (TOP PRIORITY — RETHINK)

The current carousel builder is a dialog. User wants a **full content creation studio** — a page, not a dialog.

**Key requirements (from user feedback):**
- **Browsable template library** — hundreds of pre-made carousel templates users can scroll through. NOT AI-generated suggestions. AI SEARCHES the library to find matches.
- **Works from scratch** — don't require an existing photo. User can open the builder directly, browse templates, generate a full carousel.
- **Existing photo is optional** — can be slide 1 if provided, but not required.
- **Product/reference uploads per slide** — attach product images, scene references to individual slides.
- **Reference/item library** — save products, backgrounds, outfits as reusable items.
- **Full page, not dialog** — complex creation flows need space. Dialogs feel "weak."
