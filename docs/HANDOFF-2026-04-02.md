## Architecture Decisions

- **Kling 3.0 for all video** — best character consistency via Fal.ai
- **Kling lip sync** ($0.014/unit) — much cheaper than Hedra (not on Fal.ai)
- **ElevenLabs Turbo v2.5** for voice — industry standard
- **Grok Fast for prompt enhancement** — proven in wizard, now in content gen
- **References are universal input** — same refs work for all content types
- **Factory model** — studio is a content factory, types are product lines
- **Canva-style UX** — visual-first, not form-first
- **Library not "Your World"** — single panel with refs + public + recent
- **Opt-in configs** — empty defaults, no forced styles
- **Auto-enhance prompts** — user types casual, system makes it detailed
