## Known Issues / Polish Needed

### Critical
1. **Browser panel too cramped on smaller screens** — needs collapsible/overlay behavior like Canva. Currently fixed at 280px/220px.
2. **Studio should be its own page** (`/workspace/studio`) — currently an overlay that loses URL state. Users refreshing lose context (sessionStorage helps but isn't ideal).

### Important
3. **No drag-and-drop upload in browser** — upload only via button/dialog
4. **Prompt enhancement visibility** — user wants option to control enhancement level, possibly as paid feature
5. **Template referenceKeys not wired** — templates with baked-in refs don't auto-attach them
6. **No pagination in content browser** — loads all content at once, no "Load more"
7. **1024px tablet breakpoint missing** — browser should start collapsed on tablets

### Minor
8. **Carousel action buttons could look better** — layout in canvas is functional but not polished
9. **No keyboard shortcuts** beyond Esc
10. **No download with metadata stripping** — download link is direct S3 URL, should strip AI metadata
