to get this:

![openart-image_iFf4D8Us_1769721892097_raw (1).png](https://assets.skool.com/f/50266cea381e4660ae9b2945179d86e1/797d9467e837417fa7e7bbcd32996cccc0f82d5af7394bf988e4d3a36683dee4-md.png)Once I was happy with the amount of images I got I simply run them all through Kling 2.6 with very simple actions but making sure that every prompt includes: 'camera handheld movement with visible motion'

Once I had all the videos, I simply put them all inside Capcut/Edits and started clipping it so changing of the scenes happen on every beat drop.

Every single vlog I did so far I used the same workflow.
