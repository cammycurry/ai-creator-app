If you wanna create a close up of her face from a reference image where her face is not fully visible you can upload that reference image together with your base image in the omni reference box and type the prompt like this:

''close up of her face, raw iphone photography, face like on image 2, everything else like on image 1''

![openart-image_l4Rh117T_1771691439814_raw.png](https://assets.skool.com/f/50266cea381e4660ae9b2945179d86e1/17fe09df09984a3cabf290ed731d90dac5bc9bfbf8e24ee5bd1966b6a5ebcc42-md.png)Once you have a set of images ready you just run them all through Kling 3.0 for 3 seconds each.

For maximum realism in terms of movement so it looks like someone is recording her handheld with an iphone make sure to add what you want her to do PLUS this in the prompt this:

''camera handheld with visible motion like recorded on an iphone handheld''
