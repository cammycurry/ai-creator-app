Voice Mode — Arcads Pattern Applied to Ours

How the Arcads actor-card + settings-panel pattern works when adapted for our Creator-centric model. Click the gear icons to toggle the right panel.

Arcads Their pattern

  • Multiple stock actors per video — pick from 300+
  • Each actor has a generic voice with audio settings
  • Script box + inline emotion tags (/curious)
  • Actor cards below script — play, gear, trash
  • Gear on actor → per-actor Audio Settings (right panel)
  • Settings icon in toolbar → global Actors Settings (right panel)
  • Bottom bar: actor count, add emotions, char count, settings, send

Ours Our adaptation

  • Active Creator auto-loaded — Sophia is always there
  • Each Creator has a persistent custom voice profile
  • Same script + emotion pattern works great
  • Creator cards replace actor cards — same play/gear/trash
  • Gear on Creator → per-Creator Voice Settings (right panel)
  • Settings icon → global Generation Settings (right panel)
  • Can add additional Creators for multi-person videos
  • Bonus: AI can write the script for you

Key Differences from Arcads
Creator auto-loads
Arcads makes you browse 300+ actors. We pre-load your active Creator. Sophia's voice is always ready — just write and go.
Persistent voice profiles
Arcads actors have generic voices. Our Creators have custom voice profiles you set up once — consistent across every video.
AI writes scripts
Arcads: you write the script yourself. Us: "write me a 30s script about morning workouts" in Chat mode, AI writes it, then switch to Voice to generate.
